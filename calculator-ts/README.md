# Таможенный калькулятор РФ (авто)

CLI‑утилита для расчёта растаможки легковых автомобилей по ставкам РФ.

- Ставки пошлин/утиль/акциз/сбор хранятся в SQLite `customs_rates.db`.
- Курс валют берётся с ЦБ РФ (`https://www.cbr-xml-daily.ru/daily_json.js`), при отсутствии интернета — заглушка.
- Таблицы ставок соответствуют публичным калькуляторам и нормативке на 2025 г. (см. `trans.html`).

## Быстрый старт

1) Установите зависимости:
```
pip3 install -r requirements.txt
```

2) Запустите калькулятор:
```
python3 main.py
```

3) Опционально, можно переинициализировать БД дефолтами из меню (п.2) или обновить ставки (заглушка, п.3).

## Файлы
- `main.py` — CLI, классы `RatesFetcher`, `RatesStore`, `CustomsCalculator`.
- `rates_updater.py` — заглушка обновления ставок (место для парсинга оф. источников).
- `trans.html` — справочные таблицы ставок.
- `customs_rates.db` — БД ставок (создаётся автоматически).

## Примечание
Расчёт без стоимости доставки по умолчанию. Внутри калькулятора есть опциональный блок расчёта доставки (если потребуется).
