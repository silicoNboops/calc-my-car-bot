# Руководство по гибридным автомобилям в таможенном калькуляторе

## Упрощенная логика выбора типа двигателя

Калькулятор использует простую логику определения типа автомобиля:

### Обычные типы (НЕ гибриды):
- **`gasoline`** - обычный бензиновый ДВС
- **`diesel`** - обычный дизельный ДВС  
- **`electric`** - электромобиль

### Гибридные типы (автоматически гибриды):
- **`diesel_electric`** - дизельный и электрический гибрид
- **`gasoline_electric`** - бензиновый и электрический гибрид

## Параметры для гибридов

При выборе гибридного типа (`diesel_electric` или `gasoline_electric`) запрашиваются только:

1. **Силовая установка последовательного типа** (да/нет)
2. **Мощность ДВС больше максимальной 30-минутной мощности ЭД** (да/нет)
3. **Общая мощность автомобиля** (л.с.)

**Упрощения:**
- НЕТ вопроса "Гибридный?" (определяется автоматически)
- НЕТ отдельных вопросов про мощность ДВС и электрической части
- Используется только общая мощность автомобиля

### Алгоритм расчета

```
if последовательный:
    if мощность_ДВС > мощность_ЭД:
        считать_как = "ДВС"
    else:
        считать_как = "ЭЛЕКТРО"
else:
    считать_как = "ДВС"
```

## Таблица расчетов

| Сценарий | Акциз | Пошлина | НДС | Утилизационный сбор |
|----------|-------|---------|-----|-------------------|
| Последовательный, ЭД ≥ ДВС | По ЭД | 15% | 20% | 3 400 ₽ / 5 200 ₽ |
| Последовательный, ДВС > ЭД | По ДВС | По объёму ДВС | 20% | По ДВС объёму |
| Параллельный гибрид (любой) | По ДВС | По объёму | 20% | По объёму |

## Примеры использования

### Командная строка

```bash
# Последовательный гибрид, ЭД ≥ ДВС (считается как электромобиль)
python customs_calculator_v6.py --type car --importer phys_personal \
  --cost 25000 --currency EUR --age 2 --volume 2000 --power 200 \
  --engine dvs --fuel gasoline_electric --series-hybrid \
  --dvs-power 120 --electric-power 100

# Последовательный гибрид, ДВС > ЭД (считается как ДВС)
python customs_calculator_v6.py --type car --importer phys_personal \
  --cost 25000 --currency EUR --age 2 --volume 2000 --power 200 \
  --engine dvs --fuel gasoline_electric --series-hybrid --dvs-greater-electric \
  --dvs-power 120 --electric-power 100

# Параллельный гибрид (считается как ДВС)
python customs_calculator_v6.py --type car --importer phys_personal \
  --cost 25000 --currency EUR --age 2 --volume 2000 --power 200 \
  --engine dvs --fuel diesel_electric --dvs-power 120 --electric-power 100
```

### Интерактивный режим

```bash
python customs_calculator_v6.py --interactive
```

В интерактивном режиме:

**Для обычных типов** (`gasoline`, `diesel`, `electric`):
- Никаких дополнительных вопросов про гибрид

**Для гибридных типов** (`diesel_electric`, `gasoline_electric`):
1. Калькулятор автоматически запросит:
   - Силовая установка последовательного типа? (да/нет)
   - Мощность ДВС больше максимальной 30-минутной мощности ЭД? (да/нет)
2. Укажите только общую мощность автомобиля

**Максимальные упрощения:**
- Никаких вопросов "Гибридный?" - определяется автоматически
- Никаких отдельных вопросов про мощность ДВС и электрической части
- Только два ключевых параметра для определения логики расчета

## Особенности расчета

### Акциз
- **Последовательный, ЭД ≥ ДВС**: Акциз рассчитывается как для электромобиля
- **Последовательный, ДВС > ЭД**: Акциз рассчитывается как для ДВС (физлица освобождены)
- **Параллельный**: Акциз рассчитывается как для ДВС (физлица освобождены)

### Пошлина
- **Последовательный, ЭД ≥ ДВС**: Фиксированная ставка 15%
- **Последовательный, ДВС > ЭД**: По объему ДВС согласно тарифам
- **Параллельный**: По объему ДВС согласно тарифам

### НДС
- **Последовательный, ЭД ≥ ДВС**: 20% для всех категорий импортеров
- **Последовательный, ДВС > ЭД**: 20% только для юридических лиц
- **Параллельный**: 20% только для юридических лиц

### Утилизационный сбор
- **Последовательный, ЭД ≥ ДВС**: Как для электромобилей (3 400 ₽ / 5 200 ₽)
- **Последовательный, ДВС > ЭД**: По объему ДВС
- **Параллельный**: По объему ДВС
