По НК РФ ст. 193 на 2025 год установлены следующие ставки акцизов на легковые автомобили: до 90 л.с. - 0 руб., 91-150 л.с. - 61 руб./л.с., 151-200 л.с. - 583 руб./л.с., 201-300 л.с. - 955 руб./л.с., 301-400 л.с. - 1628 руб./л.с., 401-500 л.с. - 1685 руб./л.с., свыше 500 л.с. - 1740 руб./л.с.

**❗ ЛОГИЧЕСКАЯ ОШИБКА В РАСЧЕТЕ АКЦИЗА**: 

Ваша функция `calc_excise_flat()` применяет ставку ко всей мощности, что неправильно. В НК РФ акциз рассчитывается **прогрессивно** - каждая ставка применяется только к соответствующему диапазону мощности.

```python
# НЕПРАВИЛЬНО (ваш код):
def calc_excise_flat(power_hp: int) -> float:
    if power_hp <= 90:
        return 0.0
    # Найти ставку и умножить на всю мощность
    for bracket in RATES_2025['EXCISE_RATES']:
        if power_hp <= bracket['max_hp']:
            return float(power_hp) * float(bracket['rate'])
```

**Правильный расчет**: для автомобиля 200 л.с.:
- 0-90 л.с.: 90 × 0 = 0 руб.  
- 91-150 л.с.: 60 × 61 = 3 660 руб.
- 151-200 л.с.: 50 × 583 = 29 150 руб.
- **Итого: 32 810 руб.**

Ваш код дает: 200 × 583 = 116 600 руб. (завышено в 3,5 раза!)
